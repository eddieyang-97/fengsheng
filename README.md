# 风声 Web App

《风声再临》实体版实现，支持 **2 人决斗**和 **5–8 人标准局**。

## 开始游戏

打开：[https://fengsheng-9jnw.onrender.com](https://fengsheng-9jnw.onrender.com)

1. 输入名字，选择人数并创建房间。
2. 把邀请链接或六位房间码发给朋友。
   不占座位的朋友可选择“旁观房间”，游戏开始后也能加入旁观。
3. 玩家到齐后可调整座位；房主也可在空位添加 AI 玩家，再选择按当前座位或随机座位开始。
4. 游戏中按高亮的卡牌和按钮操作。服务器只会显示当前合法行动。
5. 对局内可使用聊天、常用表情、送花和扔番茄与其他玩家互动。

## 游戏提示

- 每人起始摸 2 张牌；回合开始摸 2 张。
- 进入传情报阶段前，手牌不得超过 7 张。
- 反应按座位顺序处理；没有可用反应时默认自动跳过，可取消勾选，并可在大厅或游戏中调整自己的等待时间。
- 房主可以设置反应时限，默认 15 秒。
- AI 玩家会根据公开行动、已知情报和局面收益推测其他玩家阵营，并自动完成自己的行动与反应。
- 真人玩家断线后，房主可让 AI 临时接管其座位；原玩家重连时会自动取回控制权。
- 旁观者只会看到公开牌面、公开阵营和行动记录，不会看到手牌或获得操作权限。
- 玩家和旁观者均可参与对局聊天并使用常用表情；聊天记录会在当前对局内保留。
- 玩家可点击其他玩家的头像送花或扔番茄；这些互动只播放动画，不影响游戏流程。
- 游戏提供摸牌、出牌、传递情报和烧毁等音效，可随时静音。
- 公开记录可按相关玩家筛选；玩家也可为其他玩家添加仅自己可见的推测身份标记。
- 桌面端的公开记录与聊天侧栏可拖动调整两部分的高度。
- 有玩家断线时游戏暂停；房主可将其判定死亡后继续。
- 死亡玩家不能进行游戏操作，身份会立即公开；死亡房主仍可使用房间管理功能。
- 游戏结束后，房主可选择“新游戏”，让所有玩家保留房间和座位并返回大厅。

## 当前状态

主要游戏流程、实体牌组、功能牌、反应窗口、胜负判定、房间邀请、旁观、聊天、玩家互动、AI 玩家和断线处理均已实现，可以进行朋友间试玩。

目前房间只保存在服务器内存中：服务器重启或重新部署会结束正在进行的游戏。免费服务器闲置后首次打开可能需要约一分钟启动。
